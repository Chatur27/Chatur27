## Featured work

### Moris Beauty — Interactive 3D Storefront

A fictional premium Mauritian beauty-commerce concept designed as a responsive portfolio project.

**Highlights**

- Interactive 3D product presentation
- Responsive catalogue and product pages
- Beauty Finder recommendation flow
- Shopping-cart experience
- Mauritius delivery and click-and-collect concepts
- WebGL and reduced-motion fallbacks
- Performance-aware 3D rendering

**Technology:** Next.js, React, TypeScript, Three.js, React Three Fiber, Vercel

[Live demonstration](https://moris-beauty.vercel.app/) • [Source repository](https://github.com/Chatur27/moris-beauty)

> Moris Beauty is an original fictional portfolio project and is not affiliated with Simple.mu or an existing beauty retailer.

### Aura XR — Interactive Product Configurator

A cinematic 3D product-launch experience for a fictional spatial-computing visor.

**Highlights**

- Four-stage product journey: Overview, Anatomy, Configure, and Reserve
- Interactive 3D visor presentation
- Finish, optical mode, storage, fit, and accessory selection
- Dynamic pricing and build summary
- Responsive viewport-fit interface
- WebGL recovery and accessibility safeguards

**Technology:** Next.js, React, TypeScript, Three.js, React Three Fiber

**Status:** Final refinement and deployment in progress.
